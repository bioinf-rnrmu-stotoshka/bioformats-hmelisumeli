sam_reader module

sam_reader.py

Модуль для чтения и парсинга SAM-файлов. Предоставляет класс и функцию для интерактивного считывания строк SAM и представления их в виде объектов.

sam_reader.read_sam(filepath: str) → Iterator[SAMRecord][исходный код]

Читает SAM-файл и возвращает генератор объектов SAMRecord.

Комментарии-заголовки (строки, начинающиеся с „@“) пропускаются.

Параметры:

filepath (str) – Путь к SAM-файлу.

Yields:

SAMRecord – Объект, содержащий поля одной SAM-записи.
