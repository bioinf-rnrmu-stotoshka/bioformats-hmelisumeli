vcf_reader3.read_vcf(filepath: str, include_header: bool = False) → Iterator[VCFRecord][исходный код]

Читает VCF-файл и возвращает генератор объектов VCFRecord.

Заголовочные строки (начинающиеся с „##“) пропускаются, а строка с названиями колонок („#CHROM“) опционально может быть возвращена.

Параметры:

  • filepath (str) – Путь к VCF-файлу.

  • include_header (bool) – Если True, строка заголовка колонок возвращается как VCFRecord с полями None.

Yields:

VCFRecord – Объект записи VCF или None для заголовка, если include_header=True.
