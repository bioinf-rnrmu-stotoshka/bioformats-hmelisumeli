fastacounterF module

class fastacounterF.FastaAnalyzer(fasta_file)[исходный код]

Базовые классы: object

Класс для анализа FASTA-файлов. Предоставляет методы для итерации по записям и подсчёта статистики по последовательностям.

fasta_counter()[исходный код]

Подсчитывает общее количество записей и среднюю длину последовательности в FASTA-файле.

Результат:

average_length — средняя длина последовательности; sequence_count — количество последовательностей.

Тип результата:

tuple[float, int]

fasta_sequence_generator()[исходный код]

Генератор, который построчно читает FASTA-файл и возвращает заголовок и полную последовательность для каждой записи.

Yields:

tuple[str, str] – Кортеж из заголовка (без знака „>“) и полной последовательности (строка нуклеотидов/аминокислот).
