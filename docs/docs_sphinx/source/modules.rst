Модули
======

В этом разделе перечислены все модули пакета **code_scripts**.

Пакет **code_scripts** содержит код для чтения и анализа файлов форматов FASTA, FASTQ, SAM и VCF.

Список модулей:

- `Модуль fastaCounterF <fastacounterF.html>`__ — подсчет последовательностей в файлах FASTA
- `Модуль fastq_reader <fastq_reader.html>`__ — чтение и парсинг файлов FASTQ
- `Модуль sam_reader <sam_reader.html>`__ — чтение и обработка файлов SAM
- `Модуль vcf_reader3 <vcf_reader3.html>`__ — анализ и статистика по файлам VCF
